## 조달 경로 체크리스트

### Step 1. 계약 방식 판단
- [ ] 추정가격 5천만원 미만 → **수의계약 가능**
- [ ] 추정가격 5천만원 이상 → **경쟁입찰 원칙**
  - [ ] 나라장터 등록 여부 확인
  - [ ] 제한경쟁/지명경쟁 사유 해당 여부
- [ ] 혁신조달(혁신제품/서비스) 트랙 적용 가능 여부
- [ ] 조달청 디지털서비스몰 등록 여부

### Step 2. 보안 요건 확인
- [ ] CSAP 클라우드 보안인증 (서비스 유형: SaaS / PaaS / IaaS)
- [ ] 국정원 CC인증 해당 여부
- [ ] 개인정보영향평가(PIA) 필요 여부
- [ ] 망분리 환경 예외 적용 가능 여부 (인터넷망 서비스 여부)

### Step 3. 예산 집행 가능성
- [ ] 당해연도 예산 편성 여부
- [ ] 목적사업비 vs. 일반운영비 구분
- [ ] 자본적 지출(자산) vs. 경상적 지출(비용) 분류
- [ ] 연말 집행 가능 일정 확인 (계약 → 검수 → 지출)

### Step 4. 계약 리스크
- [ ] 기술사양 규격 적합성 (특정 업체 유리 규정 금지)
- [ ] 분리발주 필요 여부 (S/W·H/W·유지보수)
- [ ] 하도급 통보 의무 여부
- [ ] 계약이행보증보험 요건
